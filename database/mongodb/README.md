# MongoDB Practice

This folder contains beginner-friendly MongoDB examples.

## Collection

Use this collection:

```text
activity_logs
```

## Example Document

```json
{
  "studentId": 1,
  "action": "Student Created",
  "timestamp": "2026-07-02T10:00:00.000Z"
}
```

## Why Use Activity Logs?

Activity logs are useful for tracking what happened in the application.

For this project, logs can be created when:
- A student is created
- A task is created
